## 专家核心思路

### 核心定位

- 陈青峰是“历史同期尾数法”里最强规则化的专家之一，核心不在原始号码，而在尾数层行为。
- 他的方法天然带有周期感和强杀号属性，和只看近10期走势的专家完全不是一类。
- 从往期样本看，他更像是在做“尾数统计决策”，号码只是尾数判断的最终映射结果。

### 分析框架

- 固定顺序是：`上期开奖 -> 历史同期五区比/尾数奇偶/尾数012路/尾数大小/后区尾数 -> 必杀一尾 -> 双胆 -> 杀8码 -> 后区杀4码 -> 15+5 -> 8+3 -> 5+2`。
- 历史同期数据是主证据层，杀号和双胆是压缩层，复式和单注是落地层。
- 他的模板最大的特点是：每一步都在压缩，不允许只讲统计、不讲执行。

### 核心分析逻辑

- 先用目标期号去抓历史同期样本，把不同年份的同期开奖放在同一口径下比较。
- 再把这些样本全部映射到尾数层，分析尾数奇偶、尾数012路、尾数大小以及五区比偏态。
- 他真正压缩的不是原始号码，而是 0-9 尾数这一层；只有尾数结论定清楚后，才把对应号码映回前区和后区。
- 当某个尾数、某一路或某个尾数大小形态明显偏冷时，就转化成 `必杀一尾` 或强杀号逻辑。
- 后区也遵循同一原则，先看历史同期后区尾数，再落成 `后区杀4码` 和重点两码。

### 出号心理和策略

- 从往期文风看，陈青峰非常相信“期号周期 + 尾数重复行为”，所以他敢先在尾数层下重判断。
- 他的策略心理不是先找热点，而是先找可以明确排除的对象，这也是他总把杀号写得很重的原因。
- 他并不追求很大的号码池，而是追求一条连贯的压缩链：先杀尾、再杀码、再定胆、再落小单。
- 当历史同期样本足够集中时，他会明显更激进，尤其在某一路或某一尾很冷时会直接重压。

### 命中一二等时的强化特征

- 从 24005、23007、21147、21111 等一等奖样本和多篇二等奖样本看，陈青峰的中奖期与普通期在方法框架上几乎完全一致，核心一直是“历史同期尾数法”。
- 中奖样本最值得盯的是压缩动作：`前区必杀一尾`、`前区杀8码`、`后区杀4码` 几乎总是同时出现，说明他真正的命中优势来自连续多层剔除，而不是单一选号灵感。
- 一等奖样本里常见更鲜明的尾数012路倾斜，例如直接把某一路压到极冷或接近断路，这比普通期更能体现他的执行强度。
- 与普通期相比，不同点不在“多了什么新方法”，而在“尾数层判断更敢下结论”；研究时要重点看他何时敢明确压掉一尾、压掉一路。

## 分析步骤

1. 写上期开奖时，先把前区奇偶比、大小比、五区比和后区路数写全，这是进入历史同期比较的起点。
2. 读取历史同期样本，先看五区累计分布，判断本期各区是该热、该断，还是该回补。
3. 把历史同期前区号码全部降维到尾数层，分别统计尾数奇偶、尾数012路、尾数大小，并明确偏态。
4. 先下 `必杀一尾`，这是整套方法第一刀；如果这一刀不够明确，就说明本期证据不足，不能硬压。
5. 在 `必杀一尾` 的基础上，再从五区比和尾数偏态中筛出 `前区双胆`，双胆必须和尾数逻辑一致。
6. 继续扩展成 `前区杀8码`，把尾数上已经偏冷、分区上又不占优的号码统一剔出。
7. 对后区重复同样的尾数压缩逻辑，形成 `后区杀4码` 和后区重点两码。
8. 最后用同一条逻辑链生成 `15+5复式推荐`、`8+3小单推荐` 和 `5+2单注推荐`，保证小池嵌在大池里。

### 步骤中的死扣细节

- 历史同期样本一定是主证据，不能临时换成近10期热点。
- `必杀一尾`、`杀8码`、`后区杀4码` 这三层是陈青峰方案的命门，缺一层就不像他。
- `必杀一尾` 是尾数层第一刀，`前区杀8码` 则是把尾数结论映回号码层后的第二刀；它可以叠加五区比、尾数012路和尾数大小去精确到 8 个号码，不等于简单“杀两个尾数”。
- 双胆必须是尾数逻辑筛完之后的结果，不是先拍脑袋选两个顺眼号码。

## 取舍规则

- 历史同期是主证据，不能被最近10期短样本替代。
- 先在尾数层做判断，再回到具体号码层，不允许直接跳到号码池。
- `必杀一尾` 是该方法的核心压缩动作，不能删。
- 如果尾数统计与五区比冲突，先保留尾数逻辑，再在对应区间内选码。
- 预测最新一期前必须先刷新 `references/replay.md`，不能直接沿用旧日期快照。

## 可迁移经验

### 核心经验总结

- 先看尾数再看号码，是陈青峰最重要的抽象动作。
- 历史同期的价值，不在于“像不像”，而在于把样本统一到同一期号节奏下。
- 历史同期是主证据，近10期只是辅助体感，不能反客为主。
- 强杀号不是附属动作，而是他方法能够落地的关键。
- 这套方法最适合研究“周期 + 尾数 + 强压缩”三者如何配合。

### 复制时的执行习惯

- 先问自己本期哪一尾最该杀、哪一路最冷、哪一区最弱。
- 只有这三件事明确了，双胆和复式才有意义。
- 陈青峰的方法最怕被写成“有同期统计，但最后出号还是随手拼”，一定要让统计直接驱动输出。

## 局限与禁忌

- 过度依赖历史同期，可能弱化对当期短线波动的感知。
- 尾数统计很强，但对连号、重号、龙头凤尾等号码结构关注不足。
- 不要把陈青峰的方法改写成“近期热号法”或“龙头凤尾法”，那会丢掉其核心辨识度。
- 历史同期样本必须真实可追溯，不能凭印象写结论。
