# 富哥大乐透方法 Skill

详见 `references/replay.md`。预测最新一期前，先运行 `python d:\ZyProject\supser\scripts\update_dlt_replays.py --skill dlt-expert-fuge` 刷新逐期记录和一二等中奖记录；`replay.md` 必须视为按当前使用日期生成的动态快照，而不是固定历史样本。
同一次刷新会同步生成 `d:\ZyProject\supser\data\dlt-history.json`、`d:\ZyProject\supser\data\dlt-support.json` 和 `d:\ZyProject\supser\data\dlt-support.md`，用来补足最近10期原始开奖、龙头凤尾、历史同期、遗漏冷热、三区分层和 460 期条件转移这些底层数据支撑。

## 使用场景

- 用户要按富哥的节奏写大乐透分析文。
- 用户要复盘一套偏“头尾锚点 + 分区结构”的前区建模方法。
- 用户想研究一种同时覆盖前区和后区、但不强调杀号的均衡型模板。

## 必读输入

- 上一期大乐透开奖号码。
- 上一期前区大小比、奇偶比、五区比、和值。
- 最近10期龙头、凤尾、奇偶、大小、五区走势。
- 上一期后区奇偶比、大小比，以及最近10期后区组合形态。
- 当日刷新后的 `references/replay.md`，用它承接最新逐期记录、推荐位历史和一二等中奖记录。
- `d:\ZyProject\supser\data\dlt-support.json` / `d:\ZyProject\supser\data\dlt-support.md`，重点读取最近10期原始开奖、五区累计、龙头凤尾序列和后区结构统计。

## 专家核心思路

### 核心定位

- 富哥是典型的“头尾锚点 + 空间分布”型专家，核心不是重杀号，而是先把号码池的边界和空间骨架搭起来。
- 他的文章看起来模块不少，但真正的主轴只有两条：一条是 `龙头/凤尾定边界`，一条是 `五区比/大小比定分布`。
- 从往期文风看，富哥更像是在做一张“结构图”，不是在做极限压缩表。

### 分析框架

- 固定顺序是：`上期结构复盘 -> 龙头 -> 凤尾 -> 奇偶 -> 五区比 -> 大小 -> 后区 -> 前区双胆 -> 15+5 -> 9+3 -> 单注`。
- 这套顺序不能随意打乱，因为富哥是先用头尾做边界，再用区比和大小做中段回填，最后才落到双胆和号码池。
- 其中 `前区双胆` 是全文的收束点，不是附带输出。

### 核心分析逻辑

- 龙头决定前区最左边界，凤尾决定最右边界，先把号码空间夹住。
- 五区比决定号码在 01-35 空间中的占位方式，是富哥最核心的“空间思维”。
- 奇偶和大小不是第一层决策，而是校正层，用来判断本期是偏热延续还是做回补。
- 在没有强烈断区信号时，他默认规避全奇、全大、单区扎堆这类极端结构，宁可让组合贴近常态分布，也不轻易押极端。
- 后区的思路也一样，不追求复杂模型，只看上期奇偶/大小形态之后，本期是延续还是回摆。

### 出号心理和策略

- 从往期样本看，富哥出号时最在意的是“号码有没有骨架”，所以总是先抓龙头和凤尾，不愿意从中段漫无目的铺码。
- 他不喜欢先杀很多号码，而是更习惯先把空间站位摆正，再通过双胆去表达主心骨。
- 他的策略心理很明显：宁可让复式池保持一定宽度，也要保证整篇文章的结构逻辑自洽。
- 真正的信心点通常落在 `头尾方向 + 五区偏态 + 双胆` 三者一致时，而不是某一个孤立指标特别亮眼。

### 命中一二等时的强化特征

- 从 24003、24047、25015 等一二等奖样本看，富哥中奖时并没有换模型，仍然是“龙头 + 凤尾 + 奇偶 + 五区比 + 大小 + 后区 + 双胆”的完整链路。
- 真正更强的地方在执行力度：中奖样本里的五区比判断通常更坚决，经常直接给出明显断区或单热区结构，而不是模糊说“均衡分布”。
- 中奖样本里 `前区双胆` 一直存在，说明富哥的命中核心之一，是先把头尾和区比落地后，再用双胆把号码池进一步定心。
- 与普通期相比，富哥不是增加杀号，而是把头尾方向、断区判断和双胆三者更强地绑在一起；研究时要重点看这三个模块是否同向发力。

## 分析步骤

1. 写上期开奖，先把前区大小比、奇偶比、五区比、和值，以及后区奇偶比、大小比写全，给后文建立统一参考坐标。
2. 做龙头判断时，不只看上期龙头是多少，还要看最近几期龙头是连续走低、连续走高，还是在某个小区间横摆，再决定本期龙头偏大还是偏小。
3. 做凤尾判断时，重点看近期凤尾是集中在 30 以上、25-30 之间，还是明显下沉；富哥会先定凤尾大方向，再谈具体号码。
4. 做奇偶分析时，用最近10期累计结果判断本期是否有奇偶回补需求，但奇偶比只能作为中段修正，不能推翻头尾判断。
5. 做五区比分析时，先判断热区、冷区和断区风险，再决定本期哪些区必须留码、哪些区可以少留码；这一步是前区空间成形的关键。
6. 做大小分析时，重点不是单纯判断大热或小热，而是看它是否与五区比同向；如果不同向，优先保留五区结构。
7. 做后区分析时，围绕上期后区是奇奇、偶偶还是奇偶，大大、小小还是大小，判断本期是延续还是回摆，并给出大范围和重点两码。
8. 把龙头、凤尾、五区比和大小比共同约束到一个前区池里，再从中提取最能代表结构中心的两码，形成 `前区双胆`。
9. 用双胆作为主心骨，向外展开成 `15+5复式`；再把同一逻辑收缩成 `9+3复式`，最后从双胆和后区重点中落一注单注。

### 步骤中的死扣细节

- 龙头和凤尾一旦定完，后面的中段号码不能再把边界冲掉，否则就不是富哥的写法。
- 五区比的作用是定“每个区要不要留码、留几个”，不是简单罗列热区。
- 双胆通常不是两个最热号，而是最能把全文结构串起来的两个号。

## 取舍规则

- 龙头和凤尾是先验锚点，不能放到全文后面去补写。
- 如果五区比和大小比冲突，优先保留五区结构，再用大小比做微调。
- 只写富哥真实稳定使用过的维度，不额外加入跨度、重号、连号、杀号等模块。
- 前区双胆必须是最后输出前最明确的浓缩结论之一，不能省略。
- 预测最新一期前必须先刷新 `references/replay.md`，不能直接沿用旧日期快照。

## 可迁移经验

### 核心经验总结

- 先定边界再定中段，是富哥最值得复制的经验；这会让号码池天然更像“有结构的方案”而不是一把散号。
- 五区比不是装饰模块，而是富哥真正的中台，它负责把头尾之间的空间分配说清楚。
- 近10期短窗是他的主观察窗，长周期只做背景，不要让远期样本压过当期结构。
- 双胆必须服务整体结构，而不是只服务命中幻想；这也是他文章稳定的重要原因。
- 后区不做过度解释，反而让全文重点更集中在前区结构搭建上。

### 复制时的执行习惯

- 写作时要先问自己：本期头在哪里、尾在哪里、哪个区要断、哪个区要热。
- 如果这四个问题回答不清楚，就不要急着给号码。
- 富哥的方法本质上是“结构先行”，复制时最怕把它写成单纯热号堆砌。

## 局限与禁忌

- 富哥的体系没有稳定的杀号层，压缩成本相对高。
- 缺少历史同期与长周期冷热视角，遇到特殊节奏时容易偏依赖近10期。
- 不要替他补写复杂尾数法、012路法或长样本条件转移法。
- 该方法更像“结构化配号”，不是极限压缩法，研究时不要误判成杀伤型模型。
