# 江川大乐透方法 Skill

详见 `references/replay.md`。预测最新一期前，先运行 `python d:\ZyProject\supser\scripts\update_dlt_replays.py --skill dlt-expert-jiang-chuan` 刷新逐期记录和一二等中奖记录；`replay.md` 必须视为按当前使用日期生成的动态快照，而不是固定历史样本。
同一次刷新会同步生成 `d:\ZyProject\supser\data\dlt-history.json`、`d:\ZyProject\supser\data\dlt-support.json` 和 `d:\ZyProject\supser\data\dlt-support.md`，其中 `jiangChuanPatternSupport` 专门承接江川所需的形态跟随数据。

## 使用场景

- 用户要按江川的方式写一篇大乐透方案。
- 用户要研究一种“上期形态 -> 最近5次同形态后的下期”式的大乐透跟随法。
- 用户需要一套前区只抓两条主线、后区只抓 exact 组合，但输出层依然完整的模板。

## 必读输入

- 上一期大乐透开奖号码，以及前区大小比、奇偶比，后区大小比、奇偶比。
- 当日刷新后的 `references/replay.md`，用它承接江川大乐透往期文章、推荐位历史和一二等中奖记录。
- `d:\ZyProject\supser\data\dlt-support.json` / `d:\ZyProject\supser\data\dlt-support.md`，重点读取 `jiangChuanPatternSupport`。
- 若需要核对完整历史开奖，再读 `d:\ZyProject\supser\data\dlt-history.json`。

## 专家核心思路

### 核心定位

- 江川大乐透是典型的“一步跟随法”，核心不是看近10期综合面，而是看“上期出现了某种形态后，下期通常会怎么走”。
- 他把前区压缩到两条主线，后区压缩到一条主线，方法非常窄，但执行感很强。
- 从往期文章看，他更像是在做“形态触发后的下期复现”，不是传统走势派。

### 分析框架

- 固定顺序是：`开奖回顾 -> 前区大小比分析 -> 前区奇偶比分析 -> 后区分析 -> 前区两胆 -> 前区杀号 -> 后区五码 -> 后区三码 -> 大复式 -> 小复式 -> 单注`。
- 前区大小比和前区奇偶比是两条并列主线，二者都必须带历史同形态后的下期样本。
- 后区必须围绕上期 exact 两码组合做跟随，不能混成别家的遗漏或和值法。

### 核心分析逻辑

- 前区大小比逻辑是：先找到上期相同大小比在历史上出现过多少次，再看最近5次同形态之后，下期大小比总共偏向哪边。
- 前区奇偶比逻辑完全一致，只是把条件换成奇偶比；这两条线各自产生两码关注。
- 后区逻辑更直接：上期 exact 组合在历史上再出现时，下一期后区大多会往哪些码或哪些结构移动。
- 最终 `前区两胆` 不是随手选两个热号，而是从“大小比两码”和“奇偶比两码”里各取其一，让两条主线同时落地。

### 出号心理和策略

- 江川的出号心理非常朴素，他更相信“同形态之后会有相似跟随”，而不是相信复杂的多维结构。
- 他不喜欢把证据链铺得太宽，而是喜欢拿一个非常清楚的条件去追它的下期后果。
- 他的策略是：前区用两个最硬的形态条件定出双胆，再用杀号把池子压瘦，后区则用 exact 组合跟随给出辅助落点。
- 真正的信心点来自三条线是否同向，尤其是前区大小比和奇偶比是否同时给出较清晰的回补或延续结论。

### 命中一二等时的强化特征

- 可访问的一等奖样本是 19124 期，二等奖样本包括 26027、26012、25141 等；这些中奖样本都没有换模型，仍然是标准的“大小比跟随 + 奇偶比跟随 + 后区组合跟随”。
- 中奖样本最关键的执行动作，不是增加更多维度，而是让前区两条主线和后区一条主线同时收紧。
- 江川最有辨识度的收缩动作是：大小比先给两码，奇偶比再给两码，最后各取其一做成前区两胆；这一步在中奖样本里很稳定。
- 与普通期相比，中奖样本里更常见的是“大小回补或继续热出”和“奇偶回补或继续热出”同时明确，而不是两条线互相打架。

## 分析步骤

1. 写开奖回顾时，要明确上期前区大小比、奇偶比，以及后区大小比、奇偶比，因为这三个形态就是后文的全部触发器。
2. 做 `前区大小比分析` 时，读取 `jiangChuanPatternSupport.frontSizeFollow`，先写历史出现次数，再逐条列最近5次同样大小比开出后的下期样本。
3. 把这5次下期样本的大小比累计起来，判断本期更像继续偏大、继续偏小，还是回到均衡，并据此给出两码重点关注。
4. 做 `前区奇偶比分析` 时，读取 `jiangChuanPatternSupport.frontOddEvenFollow`，重复同样的“出现次数 -> 最近5次样本 -> 累计比值 -> 两码关注”链路。
5. 做 `后区分析` 时，读取 `jiangChuanPatternSupport.backPairFollow`，围绕上期 exact 后区组合写出现次数、最近5次同组合后的下期样本、后区大范围关注和两码落点。
6. 把大小比主线的两码和奇偶比主线的两码分别摆出来，观察哪两个号码最能同时代表两条主线，再组成 `前区两胆参考`。
7. 在不破坏本期目标大小比和奇偶比的前提下，输出 `前区杀号参考`，把与两条主线相冲突的号码剔出去。
8. 用两胆、杀号和后区大范围关注共同生成 `后区五码参考`、`后区三码参考`、`大复式参考` 和 `小复式参考`。
9. 最后让 `单注参考` 的前区和后区都嵌在前面的压缩池里，确保单注不是脱离主链的独立一注。

### 步骤中的死扣细节

- 两条前区主线都必须列“最近5次同形态后的下期样本”，这是江川最硬的证据链。
- 如果某个形态历史上可用样本不足 5 次，就如实写出现次数和实际可用样本数，并使用全部可用样本继续分析，不能因为凑不满 5 条就中断整条判断链。
- `前区两胆` 必须保留“从两条主线各取其一”的味道，不能写成两个随意热号；两条主线即使没有重复号码，也不要求强行求交集，而是分别保留各自主线里最能代表当期方向的一枚号码。
- `前区杀号参考` 不是别家那种大规模杀号，它服务的是两条主线的结构一致性。
- 后区一定要坚持 exact 组合跟随，哪怕样本数不多，也不能偷换成别的逻辑。

## 取舍规则

- 必须先写 `前区大小比分析`，再写 `前区奇偶比分析`，最后写 `后区分析`，顺序不要打乱。
- 大小比和奇偶比都必须带“历史出现次数 + 最近5次同形态后的下期样本 + 累计比例判断”，不能只写一句主观看法。
- `前区两胆参考` 要保留“从两条主线各取其一”的味道，不要直接写成任意两个热号。
- 后区必须坚持 exact 组合跟随，不要替江川补写遗漏冷热、后区和值、后区尾数等别家的模块。
- 不要把江川大乐透误写成近10期综合走势图模板；他的核心是一步跟随，不是面状结构统计。
- 预测最新一期前必须先刷新 `references/replay.md` 和共享数据文件，不能直接沿用旧日期快照。

## 可迁移经验

### 核心经验总结

- “上期形态 -> 下期跟随”是江川大乐透最值得学的思维骨架，它让判断链非常短但非常硬。
- 两条前区主线各先给两码，再收成两胆，是这套方法最有辨识度的压缩动作。
- 后区只抓 exact 组合跟随，会让后区判断异常干净，不容易被太多杂项干扰。
- 这套方法特别适合证据集中、想快速形成短链路方案的场景。

### 复制时的执行习惯

- 每次写作先问自己：上期大小比会把下期往哪边推，上期奇偶比会把下期往哪边推，这两条线有没有同向。
- 如果两条前区主线彼此冲突，就不要假装双胆很稳。
- 江川大乐透最怕被写成普通走势综述，复制时一定要守住“条件触发 -> 最近5次样本 -> 累计倾向 -> 两码”这条链。

## 局限与禁忌

- 这套方法维度非常窄，几乎不覆盖区间、龙头凤尾、跨度、和值、连号、重号、历史同期等模块。
- 如果大小比跟随和奇偶比跟随出现明显冲突，整套方法会变得难收束。
- 后区依赖 exact 组合跟随，样本数有时不大，判断容易偏锋。
- 不要把江川大乐透改写成富哥、陈冰、陈青峰那类多模块模板，也不要和江川排列三的位置拆解模板混用。
