- 用户要按文新林的方式写一篇均衡型大乐透方案。
- 用户要研究“三区 + 龙头凤尾 + 跨度 + 后区形态”的结构派模板。
- 用户需要一个前后区都覆盖、输出层完整、但不依赖杀号的稳定模板。

## 专家核心思路

### 核心定位

- 文新林是典型的“均衡结构派”，他不追求最强杀号，而是追求一套能稳定把前区和后区都讲圆的框架。
- 他的文章像一份标准结构报告：先讲三区，再讲头尾，再讲大小和跨度，最后再把后区形态接上。
- 从长期写法看，他的目标不是制造夸张结论，而是让整篇方案在结构上始终自洽。

### 分析框架

- 固定顺序是：`上期回顾 -> 三区走势 -> 龙头凤尾 -> 大小分析 -> 跨度分析 -> 后区奇偶/大小 -> 后区重点关注 -> 前区12码 -> 后区5码 -> 大复式 -> 小复式 -> 单注`。
- 前区四件套必须完整出现，尤其 `三区走势` 和 `龙头凤尾` 是骨架，`大小` 和 `跨度` 是修正层。
- 后区不单独发散，只承担收口职责，最后让整篇文章落成多层输出。

### 核心分析逻辑

- 三区走势负责判断号码在空间上的主分布，是最先决定“本期哪里该多留、哪里该少留”的模块。
- 对文新林来说，趋势判断的目标不是把每个维度都算满，而是先锁定三区主分布，再让头尾、大小、跨度围着这个主分布服务。
- 龙头凤尾负责给前区号码池加边界，没有头尾锚点，三区判断就容易散。
- 大小分析负责修正三区内部的号码倾向，而跨度分析负责控制 5 个前区号的整体张力。
- 后区再用奇偶比和大小比做简单形态判断，让前后区逻辑保持同样的“结构先行”风格。

### 出号心理和策略

- 文新林的出号心理很稳，他更像是在搭一副结构骨架，而不是用单个亮点去赌。
- 他不喜欢在正文里下太多极端杀结论，而是通过三区偏态、头尾方向和跨度落点慢慢把号码池收紧。
- 他的策略是先把前区结构搭稳，再让后区做补充，所以后区通常短而直接。
- 真正的信心点通常来自三区偏态、龙头凤尾方向和跨度判断三者一致，而不是只看某一个模块单独很强。

### 命中一二等时的强化特征

- 从 26031 的一等奖样本和 25128、25002 的二等奖样本看，文新林中奖时依然使用标准框架，没有临时增加杀号、胆码或尾数模块。
- 中奖样本里更值得注意的是三区走势经常给出更偏态的区间比，而不是温和均衡值，这说明他的命中点更像“空间倾斜判断”而不是“多维堆叠”。
- 龙头凤尾在中奖样本里通常更明确，往往不是泛泛列范围，而是很快收束到单个重点边界号，再用跨度去约束整体张力。
- 与普通期相比，文新林的差异主要是“更敢让三区失衡、头尾收紧、跨度单点化”；研究时要重点看这三者有没有形成同向结构。

## 分析步骤

1. 写上期回顾时，先把前区号码、后区号码、三区比、大小比和后区形态写全，建立本期分析的基准面。
2. 做三区走势时，要把最近10期三区累计分布、热区、冷区和可能断区全部交代清楚，再明确本期区间比倾向。
3. 做龙头凤尾时，不只是报一个范围，而是要看近期头尾是连续抬升、连续回落还是横摆，再给出更接近本期的边界号。
4. 做大小分析时，要先判断最近10期大数和小数谁占优，再看这股趋势与三区判断是否一致，用来修正区内选号方向。
5. 做跨度分析时，要把近10期跨度区间、上期跨度和常见落点连起来看，最后落成“本期跨度在某个值附近开出”的明确判断。
6. 做后区分析时，用最近10期后区奇偶比和大小比判断本期更可能是延续还是回摆，并且要给出具体的后区重点两码。
7. 在前区结构四件套都定完之后，再把空间、边界、大小和张力四层约束折叠成 `前区12码推荐`。
8. 后区先给 `后区5码推荐` 保留容错，再从同一逻辑收束成 `大复式推荐`、`小复式推荐` 和 `单注一注5+2`，保证单注一定嵌在前面的池里。

### 步骤中的死扣细节

- 三区走势一定要先于龙头凤尾，因为区间是大框，头尾是边界。
- 龙头凤尾不能只写范围，必须给出重点锚点号，否则就失去文新林的边界控制感。
- 跨度不是装饰项，它负责限制前区五个号拉得太散还是太紧。
- 后区虽然篇幅短，但必须同时体现奇偶和大小两层，不能只剩一句“看好某两码”。

## 取舍规则

- 前区一定要保留三区走势和龙头凤尾这两个模块，它们是文新林方法的骨架。
- 后区不做历史同期或尾数展开，保持奇偶/大小两层判断即可。
- 不要补写杀号、冷热转移或条件统计，否则会偏离其原始风格。
- 若三区走势与大小分析冲突，先保留三区配比，再用大小比微调具体号码。
- 预测最新一期前必须先刷新 `references/replay.md`，不能直接沿用旧日期快照。

## 可迁移经验

### 核心经验总结

- 先看空间分布，再看头尾边界，最后看大小和跨度，是文新林最稳定的结构节奏。
- 三区、头尾、跨度三者合在一起，会让前区号码池天然更像一张“有骨架的图”。
- 他不靠激进杀号取胜，而是靠“三区先定向、其余模块再收口”的顺序，让方案自然收紧。
- 他的方法适合做稳态分析，不靠极限杀号，也能把号码池收得相对有层次。
- 后区只做必要判断，反而能让全文重心始终停留在前区结构搭建上。

### 复制时的执行习惯

- 每次写文前先问自己：本期哪个区该热、哪个区该让位、头尾分别压在哪里、跨度大概落在哪。
- 如果这四个问题答不清楚，就不要急着给 `12码`，否则方案只会变成散号堆砌。
- 文新林的方法最怕被写成“什么都提一点但没有主骨架”，复制时要一直守住三区和头尾两个核心。

## 局限与禁忌

- 该方法没有稳定杀号层，压缩力度不如陈青峰或万妙仙。
- 缺少热冷统计与历史同期视角，对周期性解释较弱。
- 不要把它误写成“胆码派”，因为其历史文本并不稳定输出胆码。
- 研究时要把它理解为结构均衡模型，而不是极端缩号模型。
