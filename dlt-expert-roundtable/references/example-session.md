## 使用技巧

### 1. 强调对抗性

在prompt中明确要求：
```
请确保专家之间有真实的分歧和辩论，不要让所有人都同意相同的观点。
```

### 2. 控制时间

如果时间有限，可以简化流程：
```
请执行简化版圆桌会议：
- 第一阶段：独立分析（必须）
- 第二阶段：关键质疑（至少3组）
- 第四阶段：投票（必须）
- 第五阶段：共识（必须）
```

### 3. 聚焦特定问题

可以让会议聚焦特定争议：
```
请召开圆桌会议，重点讨论：
1. 08号是否应该选择（热码延续 vs 过热风险）
2. 07号是否应该选择（遗漏回补 vs 过冷风险）
```

## 常见问题

**Q: 会议时间太长怎么办？**
A: 使用简化版流程，或者只执行关键阶段

**Q: 专家观点太一致怎么办？**
A: 在prompt中强调"必须体现方法论差异"和"必须有真实分歧"

**Q: 如何评估会议质量？**
A: 检查以下指标：
- 是否有至少3个不同的号码组合
- 是否有真实的质疑和调整
- 是否体现了不同阵营的策略差异
