# 大乐透专家圆桌会议模式

## 概述

专家圆桌会议是一种创新的预测模式，通过让9位专家进行多轮辩论和协商，产生经过充分讨论的推荐结果。

## 核心特点

✅ **对抗性协作**：专家之间互相质疑和挑战
✅ **多维度验证**：从不同角度验证推荐的合理性
✅ **民主决策**：通过投票机制达成共识
✅ **透明过程**：完整记录推理和辩论过程

## 快速开始

### 1. 准备数据

确保以下文件已更新：
- `supser/data/dlt-support.json`
- `supser/predictions/_contexts/dlt/<期号>/snapshot-support.json`

### 2. 执行会议

在Kiro中输入：
```
请按照 supser/dlt-expert-roundtable/SKILL.md 为大乐透第XXXXX期召开专家圆桌会议
```

### 3. 获取结果

会议将输出：
- 完整的会议记录（包含5个阶段）
- 15+5大复式推荐
- 9+3小复式推荐
- 5+2单注推荐

## 与传统模式的对比

| 维度 | 传统单专家模式 | 圆桌会议模式 |
|------|---------------|-------------|
| 执行时间 | 快（1-2分钟） | 慢（5-10分钟） |
| 推荐多样性 | 低 | 高 |
| 推理透明度 | 中 | 高 |
| 决策可信度 | 中 | 高 |
| 适用场景 | 日常预测 | 重要期次 |

## 使用建议

### 何时使用圆桌会议

- 📅 重要期次（如节假日、大奖期）
- 🤔 对单专家推荐有疑虑时
- 📚 学习不同分析方法时
- 🔍 需要深入理解推理过程时

### 何时使用传统模式

- ⚡ 时间紧迫时
- 🎯 只需要快速结果时
- 💻 计算资源有限时
- 📊 日常批量预测时

## 常见问题

**Q: 圆桌会议的推荐更准确吗？**
A: 不一定。圆桌会议提供的是更充分讨论的结果，但彩票本质是随机的。

**Q: 可以自定义参会专家吗？**
A: 目前固定为9位专家，未来版本可能支持自定义。

**Q: 会议记录会保存吗？**
A: 会保存在 `predictions/dlt-expert-roundtable/` 目录下。
